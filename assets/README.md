# E-Commerce Mobile - Assets

Thư mục này chứa các assets cho mobile app.

## Required Assets

Các file assets cần có:

### Icons
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

### Splash Screen
- `splash.png` - Splash screen image (1284x2778px cho iOS)

## Tạo Assets

### Cách 1: Sử dụng công cụ online
- [makeappicon.com](https://makeappicon.com/) - Generate tất cả sizes
- [appicon.co](https://appicon.co/) - Icon generator

### Cách 2: Sử dụng Figma/Photoshop
- Tạo design 1024x1024px
- Export với chất lượng cao
- Đặt tên đúng và copy vào thư mục này

## Temporary Placeholder

Hiện tại đang dùng assets mặc định của Expo. Thay thế chúng bằng design thực tế của dự án.

### Màu sắc chính
- Primary: #007AFF (iOS Blue)
- Background: #FFFFFF
- Text: #000000

### Guidelines
- Icon nên đơn giản, dễ nhận diện
- Splash screen nên có branding rõ ràng
- Sử dụng PNG với transparent background cho icon
