# 🎉 Xong rồi!

## Chạy app:

```bash
cd Mobile
npm start
```

Scan QR bằng Expo Go app là xong!

## Nếu test trên điện thoại thật:

1. Tìm IP máy: `ipconfig`
2. Sửa `.env`: `EXPO_PUBLIC_API_URL=http://192.168.1.XXX:5153`
3. Restart: `npm start --clear`
